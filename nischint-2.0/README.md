# Nischint 2.0 Demo

This is a standalone demo of the new UI features.

## How to Run

1. Navigate to this directory:
   ```bash
   cd nischint-2.0
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

## Features Implemented

- **Interactive Ribbon**: Swappable icons for Money, Goal, and Home.
- **Neon Mic Button**: Custom 3D-style button with hover effects.
- **Audio Visualizer**: Simulated frequency bars.
- **Dark Theme**: Modern dark UI with neon yellow accents.
